Mis Proyectos

Una breve descripción de mis trabajos y su objetivo.

Perfume Azure | Social media

Este proyecto de fotomontaje para redes sociales presenta Azure, un perfume de alta gama que transmite elegancia y frescura. La composición busca resaltar la sensación de naturalidad, utilizando una paleta de tonos azulados que evocan serenidad y pureza. Cada elemento visual ha sido cuidadosamente seleccionado para reforzar el concepto de frescura, reflejando la esencia refrescante y sofisticada de Azure.

Gaseosa Casera | Social media

En este proyecto se creó una composición visual para Casera, una gaseosa de sabor hierbabuena. La presentación destaca por su limpieza y una paleta de tonos verdosos, con iluminaciones estratégicas que resaltan el producto. Se incluyeron hojas naturales como elemento decorativo para reforzar la sensación de frescura y naturalidad que define a esta bebida

Hambuerguesa | Social media

Este proyecto presenta una composición visual para una hamburguesa clásica, combinando elementos fotográficos y un toque dinámico. La imagen captura la jugosidad y autenticidad de la hamburguesa, con ingredientes frescos y perfectamente apilados. Las luces y sombras resaltan cada detalle, mientras que los elementos dinámicos, como salpicaduras de salsas y semillas en movimiento, aportan energía a la escena, haciendo que el producto luzca irresistible y vibrante.
